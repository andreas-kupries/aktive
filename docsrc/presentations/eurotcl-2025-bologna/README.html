Presentation

The presentation you are looking at here is based on the reveal.js framework.

To see the presentation simply open the file index.html in a browser, and then use the cursor keys to navigate left/right between slides, and, where possible, up/down.

The bottom right corner of each slide shows the available directions, and the page number of the slide currently shown.

The code-assets directory contains files important to the presentation while not being a direct part of it. These are

  • The border.tcl script performing page extractions on a set of jpeg pictures.

    The same script run in -debug mode generates presentations, like this one, with the exception of the code-assets directory and the READMEs.

  • The notes directory contains the static files which were integrated into this presentation during its generation.

  • The log of the border.tcl run creating this presentations, and the associated phase timing statistics.

See code-assets/README.* for more information.
